================================================================================
                    TEMPERATURE SENSOR (NTC)
================================================================================

    3.3V from ESP32
        │
        │
      ┌─┴─┐ 10K Pull-up
      │10K│ 5% tolerance
      └───┘
        │
        ├──────────────────► ESP32 GPIO 35 (ADC)
        │
      ┌─┴─┐
      │NTC│ 10K @ 25°C (Beta 3950)
      │10K│ Glass bead thermistor
      └───┘
        │
       GND

    ADC Voltage to Temperature (Steinhart-Hart):
    
    Vout = 3.3V × (Rntc / (10K + Rntc))
    Rntc = 10K × exp(3950 × (1/T - 1/298.15))
    
    Or simplified Beta equation:
    1/T = 1/298.15 + (1/3950) × ln(Rntc/10000)

  (PWM)   │      │          GND
          │      │
          │      ▼
         GND   GND

    IMPORTANT:
    • Use LOGIC-LEVEL MOSFET (IRLZ44N not IRFZ44N)
    • 10K pulldown keeps MOSFET OFF during ESP32 boot
    • PWM frequency: 1kHz (heating is slow response)
    • Duty cycle: 0-255 maps to 0-100% heater power
    • Add temperature safety cutoff in firmware!

================================================================================
                    QUICK TROUBLESHOOTING
================================================================================

    PROBLEM: ESP32 won't boot
    → Check: 3.3V regulator output
    → Check: GPIO 0 pulled LOW for upload mode
    → Check: No shorts on power rails
    
    PROBLEM: pH readings erratic
    → Check: Probe fully submerged
    → Check: Calibrate at pH 7 first
    → Check: Shielded cable for pH probe (noisy!)
    
    PROBLEM: Temperature wrong
    → Check: Beta value in code (3950 for common NTC)
    → Check: 10K pull-up resistor value
    → Check: No dry contact on thermistor
    
    PROBLEM: No electrochemical signal
    → Check: All 3 electrodes in solution
    → Check: LMP91000 configured (bias voltage set)
    → Check: Electrodes not touching (short)
    → Check: Graphite leads have good electrical contact
    
    PROBLEM: Heater won't turn on
    → Check: MOSFET is IRLZ44N (logic level, NOT IRFZ44N)
    → Check: 12V present at heating pad
    → Check: PWM signal on GPIO 4 with scope/multimeter
